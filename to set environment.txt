## **Troubleshooting Common Issues**

### **1. Java Not Found**
```powershell
# Set JAVA_HOME environment variable
setx JAVA_HOME "C:\Program Files\Eclipse Adoptium\jdk-17.x.x-hotspot"
```

### **2. PySpark Import Error**
Make sure `findspark` is installed and initialized in notebook:
```python
import findspark
findspark.init()
```

### **3. Dashboard Won't Open**
- Check port 8461 isn't in use
- Try changing `PORT = 8461` to `PORT = 8888` in the code [ppl-ai-file-upload.s3.amazonaws](https://ppl-ai-file-upload.s3.amazonaws.com/web/direct-files/attachments/9024651/42aec375-7a66-4f4f-8a32-05f6f7aa7545/bottle_neck_display.ipynb)
